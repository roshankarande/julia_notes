## How to add a package

Julia provides a large package library. To add a package, we enter the package REPL by pressing ] and install a package by the keyword add.

```julia
# Press ] from REPL 
add Plots
add PlotlyJS
and PyPlot
```

Another way

```julia
using Pkg
Pkg.add("Plots")
```
